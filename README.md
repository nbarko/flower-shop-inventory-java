# Flower Shop Inventory Management System

A Java-based inventory management application designed for managing different categories of products within a flower shop.

## Features
- Object-oriented design using inheritance, abstraction and polymorphism
- Management of different product categories (flowers, plants, tools and accessories)
- File-based inventory data storage
- Reading and processing product information from text files
- Console-based user interaction

## Technologies
- Java
- Object-Oriented Programming (OOP)
- File I/O

## Project Purpose
This project was developed as part of the Introduction to Programming course during my Bachelor's degree in Computer Science and Web Technologies.
